# DevToolsHub

DevToolsHub is a collection of fast, simple, and developer-friendly tools.

## Features

- JSON Formatter
- More tools coming soon

## Tech Stack

- Astro
- React
- TypeScript
- Tailwind CSS

## Project Structure

````text
src/
  components/   shared reusable ui components
  features/     tool-specific logic and interactive components
  layouts/      shared page layouts
  pages/        astro routes and pages
  pages/tools/  individual tool pages
  styles/       global css
  types/        shared types

## Getting Started

```bash
pnpm install
pnpm dev
````
